# 商企通前端博客

# 使用

安装hexo
```bash
npm install -h hexo
```

将仓库拉下来
```bash
git clone https://github.com/sqt-fe/sqt-fe.github.io.git
```

安装依赖
```bash
npm install
```
或
```bash
yarn
```

新建文章
```bash
hexo new "文章名"
```
> 文章名最好使用英文

本地生成文章
```bash
hexo g
```

发布文章
```bash
hexo d
```
